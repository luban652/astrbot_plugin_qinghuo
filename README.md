## astrbot_plugin_qinghuo

- 适配pjsk的sou桑 体力恢复提醒+每日挑战打卡插件

##  安装步骤

- 方法一：
- 在 AstrBot 目录下的 data/plugins 文件夹中，创建新文件夹 astrbot_plugin_qinghuo；
- 将上述 3 个文件（main.py、requirements.txt、metadata.yaml）放入该文件夹；
- 重启 AstrBot 机器人，插件会自动加载（控制台会打印「清火插件：体力定时检查线程已启动」表示加载成功）。

- 方法二：
- copy该目录为.zip
- 到 Astrbot Weubi 中以文件形式安装；

- 方法三：
- 到 Astrbot Weubi 插件目录下以链接形式安装

##  核心指令用法

- 所有指令均以 /清火 开头，兼容群聊和私聊使用，完整指令如下：

- /清火 订阅	/清火 订阅	默认订阅（上限 100，恢复 1 / 分钟）
- /清火 订阅 [上限] [速度]	/清火 订阅 150 0.8	自定义体力上限和恢复速度
- /清火 取消	/清火 取消	取消体力提醒订阅
- /清火 设置 [数值]	/清火 设置 50	手动设置当前体力值
- /清火 状态	/清火 状态	查看体力全状态（含预计满体力时间）
- /清火 每日 完成	/清火 每日 完成	标记今日挑战完成
- /清火 每日 未完成	/清火 每日 未完成	取消今日挑战完成标记
- /清火 帮助	/清火 帮助	查看所有指令帮助

##  自动提醒功能

- 插件启动后自动运行，无需手动触发：
- 体力满提醒：每分钟检查一次，体力恢复至上限时，自动 @用户发送提醒（群聊 / 私聊均支持）；
- 每日挑战提醒：每天 22:00 自动提醒所有未完成每日挑战的订阅用户；
- 每日状态重置：每天 04:00 自动重置所有用户的「每日挑战」状态为未完成。
